---
name: Proaktivně navrhovat opakující se patterny
description: Když rozpoznám opakující se vzor v práci, navrhnout jeho automatizaci a po schválení uložit do paměti.
type: feedback
---

Pokud rozpoznám pattern, který by bylo vhodné aplikovat automaticky do budoucna, navrhnout ho uživateli. Po schválení uložit jako feedback memory.

**Why:** User chce, aby se workflow průběžně zlepšoval a nemusel opakovaně zadávat stejné instrukce.

**How to apply:** Průběžně sledovat opakující se požadavky nebo přístupy. Když se pattern zopakuje 2×+, navrhnout jeho zavedení jako pravidlo.
